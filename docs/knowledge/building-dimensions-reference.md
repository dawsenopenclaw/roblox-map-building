# Building Dimensions Reference for Roblox — Complete Guide

## Scale Fundamentals

Roblox uses a unit called a "stud." The official conversion:
- **1 stud = ~0.28 meters (~0.92 feet)**
- **Roblox character height = ~5.1 studs (head to foot)**
- For practical purposes: **1 stud is roughly 1 foot**

Use the character as your measuring stick. If something looks right next to a 5-stud-tall character, it is right.

## Positioning Tips

- **Y position is the CENTER of a Part**, not the bottom. A 2-stud-tall Part sitting on the ground at Y=0 has Position.Y = 1.
- **Foundation embedding**: Set building foundation Y so bottom is 0.5 studs below ground. Looks grounded, not floating.
- **Furniture on floors**: If floor top is at Y=5, a table (3.5 tall) has Position.Y = 5 + 3.5/2 = 6.75.
- **Wall on foundation**: If foundation top is at Y=1, a 12-tall wall has Position.Y = 1 + 12/2 = 7.
- **Overlaps**: Overlap adjacent Parts by 0.05 studs to prevent z-fighting (flickering seams).
- **Character clearance**: Doors need 7H minimum, hallways 5W minimum, stairs 4W minimum.

This reference ensures every build is proportionally correct and feels right in the Roblox world.
