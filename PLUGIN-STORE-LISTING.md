## Full Description (under 1000 chars)

Type what you want to build. ForjeGames does the rest — directly inside Roblox Studio.

Connect this plugin to your ForjeGames account. Open the editor at forjegames.com. Enter the 6-character sync code from the plugin panel. Then just describe your game: "make a mountain with a river," "add a shop GUI," "write a round timer." The AI applies changes live in your scene.

What it does:
- Generates terrain, scripts, UI, and 3D assets from text prompts
- Inserts and positions marketplace assets by name
- Sets properties, moves, deletes, and parents instances
- Full undo/redo support — nothing is permanent until you want it to be
- Real-time sync between your browser and Studio

Powered by 9 AI agents and 6 AI models.

How to start:
1. Install this plugin
2. Sign up free at forjegames.com
3. Open the editor and enter the code from the plugin toolbar
4. Start building

Free to start. No credit card required.
